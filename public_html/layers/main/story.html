FW is a framework written in php and js! the first framework ever to support and maintain both front-end and back-end.

Made to be used by anyone who is looking for a lightweight multifunctional framework, this framework is a universal solution, which can be used already after being installed. It differs from other similar frameworks in everything - be it the initial concept or the final look. FW also can boast the set of powerful features that can be used by anyone without being afraid to overload their web application.
