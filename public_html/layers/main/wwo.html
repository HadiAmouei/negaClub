what FW offers

Fast Setup
You can easily setup your project by cloning the Quick Start from our github repo.
no licences
FW is an open source framework which requires no licencing.
attachment free
FW requires no attachment to any other host or project, with other words it is a project independent set of code!
development
The developers can feel free to comment or contribute on the FW framework.
MVC architecture
Instead of having to plan where things go, FW comes with a set of conventions to guide you in developing your application
innovative ux
FW makes it easier for implementing ux alongside it's awesome dx!
consultations
Feel free to contact us if any questions come up unanswered.
awesome tools
FW offers many useful tools like query builders, data seeder, js validation components and etc. .
