Frequently asked questions

What if a bug in the core files comes up?

The major advantage is being project independent: You can easily fix or modify even the core files. and even if you couldn't find the bug FW's tech support will fix it right away

What do I do if i want to use this framework?

You can use our Docs to start a project with FW.

What am I allowed to do with the framework?

You may build a website using the framework in any way you like. You may not claim intellectual or exclusive ownership to FW framework, modified or unmodified.
